# Persis İK

**Persis İK** - Personel ve İnsan Kaynakları Yönetim Sistemi

Modern ve kapsamlı bir İK yönetim uygulaması. Personel takibi, izin yönetimi, harcama takibi, İSG evrak yönetimi ve icra dosyası takibi gibi tüm İK süreçlerinizi tek platformda yönetin.

## Özellikler

- 👥 **Personel Yönetimi** - Çalışan bilgileri, departman ve şube bazlı organizasyon
- 📅 **İzin Yönetimi** - İzin talepleri, onay süreçleri ve izin geçmişi
- 💰 **Harcama Takibi** - Personel harcamalarının kaydı ve raporlanması
- 🏥 **İSG Evrak Yönetimi** - İş sağlığı ve güvenliği evraklarının takibi
- ⚖️ **İcra Dosyası Takibi** - Haciz ve icra süreçlerinin yönetimi
- 📊 **Raporlama** - Detaylı analiz ve raporlama araçları
- 🎨 **Özelleştirilebilir Tema** - 10 farklı renk teması
- 🤖 **AI Asistan** - Yapay zeka destekli yardımcı

## Başlangıç

Geliştirme sunucusunu başlatın:

```bash
npm run dev
```

Tarayıcınızda [http://localhost:3000](http://localhost:3000) adresini açın.

## Teknolojiler

- **Next.js 16** - React framework
- **TypeScript** - Type-safe kod
- **Tailwind CSS** - Modern stil sistemi
- **Drizzle ORM** - Veritabanı yönetimi
- **Better SQLite3** - Yerel veritabanı
- **Radix UI** - Erişilebilir UI bileşenleri
- **Google Generative AI** - AI entegrasyonu
